### Included files/folders:

* wave_matlab.zip --> contains original matlab code from  [Torrence and Compo][TC_98] and [Univ Colorado wavelet page][TC_home]
* wavelets_bams.zip --> contains Wavelet Analysis Routine [https://github.com/aaren/wavelets](https://github.com/aaren/wavelets) which is a clean implementation of the wave_matlab continuous wavelet transfer routines.
* wtc-r16.zip --> modified matlab routines originating from the wave_matlab.zip routines.  Implemented by [Aslak Grinsted][AK_Dept_Web] and discussed on his [website][Grinsted]
